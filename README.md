# CSandBox
这是一个基于 Python 的 Python 沙盒程序.
只支持单文件进行运行.

以支持的功能:
+ 基础数据结构
    + 字符串
    + int
    + float
    + bool
    + 列表
    + 字典
    + 元组
    + 切片

+ 函数
    + 位置参数
    + 闭包
    + 递归

+ 控制语句
    + if
    + for
    + while
+ 运算
    + 基础四则运算
    + 位运算
    + 逻辑运算
    + 几乎全部的 Python 支持的运算

以上足以满足基础的算法编写需求

不支持的功能:
+ 函数
    + 装饰器
    + 键传参

+ 类
+ 生成器(包括生成器表达式)
+ 协程
+ 导入包
+ 多文件
+ 等高级特性

如果想要了解更多请去看测试用例.
